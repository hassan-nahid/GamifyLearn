# Gamify Learn

## 🌐 Live

[![Live Site](https://img.shields.io/badge/LIVE-gamifylearn.com-00b4d8?style=for-the-badge&logo=google-chrome&logoColor=white)](https://www.gamifylearn.com/)

> **Learn. Compete. Connect. Win Together.**

Gamify Learn is a **gamified social learning platform** — the first of its kind where education meets gaming. Compete in challenges, earn rewards, connect with fellow learners, and level up your knowledge in a vibrant community.

## ✨ Features

| Feature                   | Description                                                                  |
| ------------------------- | ---------------------------------------------------------------------------- |
| **Social Feed**           | Share learning wins, ask questions, and engage with the community            |
| **Daily Challenges**      | Fun daily quests to earn points, maintain streaks, and compete for top spots |
| **Quizzes**               | Interactive knowledge tests with instant feedback across diverse topics      |
| **Live Competitions**     | Real-time tournaments and battles with friends and global leaderboards       |
| **Study Squads**          | Form study groups, collaborate on challenges, and grow together              |
| **Badges & Rewards**      | Unlock achievements, collect badges/avatars/banners/titles, redeem points    |
| **Skill Trees**           | Level up across multiple skill paths and track your complete journey         |
| **Leaderboard**           | Compete globally and climb the ranks                                         |
| **Premium Subscriptions** | Unlock exclusive content via SSL Commerz payment gateway                     |

## 🛠️ Built With

**Frontend** — Next.js, React, TypeScript, Tailwind CSS, Framer Motion, Radix UI  
**Backend** — Node.js, Express, TypeScript, MongoDB (Mongoose), Redis  
**Auth** — Passport.js (Google OAuth + Local), JWT, bcrypt  
**Payments** — SSL Commerz  
**Media** — Cloudinary  
**Email** — Nodemailer  
**Validation** — Zod  
**Scheduling** — node-cron

## 🎯 Mission

To revolutionize online education by creating an engaging, interactive, and social learning environment — where learning is fun, collaborative, and rewarding.
